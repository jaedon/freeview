참고자료.

NCP 확장자의 파일은 NClass(http://nclass.sourceforge.net/)라는 UML drawing tool을 통해서
작성된 OIP Class diagram이다.

실제 Free UML Diagram Tool로 유명한것은 StarUML(http://staruml.sourceforge.net)이지만..

우선 diagram의 look & feel 자체가 NClass가 더 좋아서 이것을 사용한 것으로 이해한다.

Class Diagram 관계도를 보고 싶은분은 위의 URL에서 Tool을 download받아서 사용하시길 바란다.
