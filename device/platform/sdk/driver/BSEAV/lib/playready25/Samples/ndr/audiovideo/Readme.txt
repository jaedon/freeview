Files in this directory:
- DotNetCommercial_500kbps_30fps.wmv: Clear (unprotected) audio/video media file
- LicenseRequestMessage.bin: License request message sent to the WMDRM-ND transmitter requesting the license
- LicenseResponseMessage.bin: The license response message from the transmitter as a response to the request above
- Keys.txt: The encryption and integrity keys extracted from the license response message.
- DotNetCommercial_500kbps_30fps.WMDRM-ND.wmv: The WMDRM-ND-encrypted file, returned by the transmitter.
